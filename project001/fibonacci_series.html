Program to print first n Fibonacci Numbers


Description
    In case of fibonacci series, next number is the sum of previous two numbers for example 0, 1, 1, 2, 3, 5, 8, 13, 21 etc. The first two numbers of fibonacci series are 0 and 1.
  • C
  • JAVA 
                    
// C program to print first n Fibonacci numbers
  #include <stdio.h>

  // Function to print first n Fibonacci Numbers
  void printFibonacciNumbers(int n)
  {
      int f1 = 0, f2 = 1, i;

      if (n < 1)
          return;

      for (i = 1; i <= n; i++)
      {
          printf("%d ", f2);
          int next = f1 + f2;
          f1 = f2;
          f2 = next;
      }
  }

  // Driver Code
  int main()
  {
      printFibonacciNumbers(7);
      return 0;
  }
OUTPUT
1 1 2 3 5 8 13
